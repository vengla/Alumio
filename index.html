🚀 Quick Start Instructions

1. Start Backend Server (Terminal 1):

cd backend npm run seed # First time only npm run dev # Starts on http://localhost:5000

2. Start Web Application (Terminal 2):

cd web npm run dev # Starts on http://localhost:3000

3. Access Applications:

Web: http://localhost:3000 Mobile: Click "Mobile Application" card above API: http://localhost:5000
